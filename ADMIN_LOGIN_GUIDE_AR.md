# 🔐 دليل تسجيل الدخول إلى لوحة تحكم المشرف

## 📋 بيانات تسجيل الدخول الافتراضية

بعد تشغيل سكريبت إنشاء حساب المشرف الأول، ستكون بيانات الدخول كالتالي:

### 👤 الحساب الافتراضي:
```
اسم المستخدم: admin
كلمة المرور: Admin@123456
الاسم: مدير النظام
الصلاحية: مسؤول رئيسي (Super Admin)
```

⚠️ **مهم جداً:** يجب تغيير كلمة المرور فوراً بعد أول دخول!

---

## 🚀 كيفية إنشاء حساب المشرف الأول

### الطريقة 1: باستخدام السكريبت التلقائي (موصى بها)

1. **افتح Terminal/CMD** في مجلد المشروع:
```bash
cd "c:\Users\ALBASHA CENTER\Desktop\مشاريع\musharaka-sa\musharaka-sa"
```

2. **تأكد من تثبيت المكتبات:**
```bash
npm install
```

3. **تأكد من وجود ملف `.env`** في مجلد `backend` أو `api` مع البيانات التالية:
```env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key-here
```

4. **شغّل السكريبت:**
```bash
node scripts/create-first-admin.js
```

5. **ستظهر رسالة النجاح** مع بيانات الدخول

---

### الطريقة 2: باستخدام SQL مباشرة في Supabase

1. **افتح لوحة تحكم Supabase** الخاصة بمشروعك

2. **اذهب إلى SQL Editor**

3. **انسخ والصق الكود التالي:**

```sql
-- إنشاء حساب مشرف رئيسي
INSERT INTO admins (username, password, name, role) 
VALUES (
  'admin',
  '$2a$10$rN8VqhK5JzKxH9L2mP3wOeYtR6sU7vW8xZ9aB1cD2eF3gH4iJ5kL6',
  'مدير النظام',
  'super_admin'
);
```

4. **اضغط Run** لتنفيذ الاستعلام

5. **تم!** يمكنك الآن تسجيل الدخول

---

## 🌐 كيفية الوصول إلى لوحة التحكم

### العنوان:
```
http://localhost:3001/admin/
```

### خطوات الدخول:

1. **افتح المتصفح** (Chrome, Firefox, Edge)

2. **اكتب العنوان:** `http://localhost:3001/admin/`

3. **أدخل بيانات الدخول:**
   - اسم المستخدم: `admin`
   - كلمة المرور: `Admin@123456`

4. **اضغط "دخول"**

5. **ستفتح لوحة التحكم** ✅

---

## 🛡️ تغيير كلمة المرور

بعد أول دخول، يُنصح بتغيير كلمة المرور:

1. **في لوحة التحكم**، اذهب إلى **الإعدادات** ⚙️

2. **اختر "تغيير كلمة المرور"**

3. **أدخل:**
   - كلمة المرور الحالية: `Admin@123456`
   - كلمة المرور الجديدة: (اختر كلمة قوية)
   - تأكيد كلمة المرور الجديدة

4. **احفظ التغييرات**

---

## 🔑 مواصفات كلمة المرور القوية

- ✅ 8 أحرف على الأقل
- ✅ تحتوي على أحرف كبيرة وصغيرة
- ✅ تحتوي على أرقام
- ✅ تحتوي على رموز خاصة (!@#$%^&*)

**مثال:** `Musharaka@2024!Secure`

---

## 👥 إدارة المستخدمين (للمسؤول الرئيسي فقط)

بصفتك **مسؤول رئيسي (Super Admin)**، يمكنك:

### إضافة مديرين جدد:
1. اذهب إلى **"إدارة المديرين"** في القائمة الجانبية
2. اضغط **"إضافة مدير جديد"**
3. أدخل البيانات:
   - الاسم
   - اسم المستخدم
   - كلمة المرور
   - الدور (مسؤول / مسؤول رئيسي)
4. اضغط **"حفظ"**

### حذف مدير:
1. في قائمة المديرين
2. اضغط على أيقونة الحذف 🗑️ بجانب المدير
3. أكد العملية

## ❓ حل المشاكل الشائعة

### المشكلة: "بيانات الدخول غير صحيحة"

**الحلول:**
1. تأكد من تشغيل السكريبت بنجاح
2. تحقق من حالة الخادم (يجب أن يكون شغالاً على port 3001)
3. تأكد من كتابة البيانات بشكل صحيح (حساس لحالة الأحرف)

---

### المشكلة: "لا يمكن الوصول إلى لوحة التحكم"

**الحلول:**
1. تأكد من تشغيل الخادم:
```bash
cd backend
npm start
```

2. تحقق من أن الخادم يعمل على `http://localhost:3001`

3. تأكد من عدم وجود جدار حماية يمنع الاتصال

---

### المشكلة: "خطأ في قاعدة البيانات"

**الحلول:**
1. تحقق من ملف `.env`:
```env
SUPABASE_URL=رابط_مشروعك
SUPABASE_SERVICE_ROLE_KEY=المفتاح_الخاص
```

2. تأكد من أن جدول `admins` موجود في Supabase

3. راجع سجل الأخطاء في Terminal

---

### المشكلة: نسيت كلمة المرور

**الحل 1 - من خلال SQL:**
```sql
-- استبدل YourNewPassword بكلمة مرور جديدة
UPDATE admins 
SET password = '$2a$10$newHashedPasswordHere'
WHERE username = 'admin';
```

**الحل 2 - إنشاء مدير جديد:**
```bash
node scripts/create-first-admin.js
```
(مع تعديل البيانات في السكريبت)
